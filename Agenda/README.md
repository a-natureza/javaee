# Dynamic Web Project
## Java Server Pages 

- Usar Servlet
- JSP para interface de usuário
- Servelts para a lógica do sistema

Scriplet:     `<%        %>`
Comentários:  `<%--    --%>`
Diretivas:    `<%@       %>`
Declarações:  `<%!       %>`
Expressões:   `<%=       %>`


### Tecnologias
- Servidor Tomcat
- Java EE - Servelet e JSP
- IDE Eclipse
- MySQL (CRUD)
- MVC (Model View Controller)
- JavaBeans
- JDBC (Java Database Connectivity)
- iText
